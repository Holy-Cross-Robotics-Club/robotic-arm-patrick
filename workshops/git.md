#GitHub Workshop

##Push
Pushing is essentially used to 'push' any changes you have made to the source code. Any changes you make to the source code can
be pushed using the command structre below in your terminal

'git push REMOTE-NAME BRANCH-NAME'


##Pull
Pulling is essentially used to 'pull' any changes that have been made to the source code. To make sure evertyhting is up to date,
it is good practice to pull before working on any code.

'git pull REMOTE-NAME BRANCH-NAME'


##Clone
Typing the below command into your terminal will locally clone our project onto your machine

'git clone https://github.com/Holy-Cross-Robotics-Club/robotic-arm-patrick.git'

##Branches
Branches are a fundamental feature in Git and GitHub that allow for parallel development and safe experimentation within a project.

How to create a git branch?
'git branch BRANCH-NAME'

How to check what branch you are in?
'git checkout BRANCH-NAME'
